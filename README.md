## Hướng dẫn cài đặt thư viện

Để chạy được mã nguồn trong file `main.py`, hệ thống của bạn cần cài đặt một số thư viện Python bên ngoài. Các module như `math` và `os` là thư viện chuẩn (built-in) của Python nên không cần phải cài đặt thêm.

### 1. Cài đặt nhanh qua Terminal / Command Prompt

Bạn hãy mở Terminal (nếu dùng VS Code, bạn có thể nhấn `` Ctrl + ` ``) hoặc Command Prompt, sau đó chạy câu lệnh sau:

```bash
pip install requests folium geopy python-dotenv
